# SOAL
*Untuk soal berikut harap discreenshot jelas & lengkap tidak ada yang terlewat
1. Impor database dari link berikut : 
2. Masuk database dengan nama PontekindoMart lalu tambah kolom harga pada tabel produk, dan isi harga pada masing-masing produk, ubah kolom kategori menjadi enum('snack','roti','minuman','permen'), masukkan 5 data tambahan ke dalam tabel produk
3. Ubah harga data id ke-5 menjadi 7000, hapus data dengan harga di atas 5000, tampilkan data dengan kategori minuman dan snack yang harganya di bawah 5000 batasi tampil cukup 3 data saja
4. Tampilkan jumlah data, harga termurah, harga tertinggi, rata-rata, dan jumlah total semua harga!
5. Ekspor database menggunakan mysqldump dan upload ke repo github
